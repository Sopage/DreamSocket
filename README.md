# DreamSocket
Socket连接客户端，精简版本的Netty或者Mina
### 使用方式

    Gradle:
    compile 'com.github.spacetimeme:dream-socket:1.0.0'
    
    Maven:
    <dependency>
      <groupId>com.github.spacetimeme</groupId>
      <artifactId>dream-socket</artifactId>
      <version>1.0.0</version>
    </dependency>
