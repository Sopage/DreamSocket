# DreamSocket
Socket连接客户端，精简版本的Netty或者Mina
### 使用方式

Gradle

    compile 'com.github.spacetimeme:dream-socket:3.0.0-SNAPSHOT'
    
Maven

    <dependency>
      <groupId>com.github.spacetimeme</groupId>
      <artifactId>dream-socket</artifactId>
      <version>3.0.0-SNAPSHOT</version>
    </dependency>
